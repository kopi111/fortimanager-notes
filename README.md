# FortiManager 7.6 — Study Notes

Personal study notes for the **FortiManager 7.6 Administrator** course
(NSE 5 / FCP_FMG_AD-7.6). Concepts only, paraphrased in my own words for
revision.

Live site: https://kopi111.github.io/fortimanager-notes/

## Topics covered

- What FortiManager is and what it does
- Inside FortiManager — modules and panes
- Administrative Domains (ADOMs), normal vs advanced device modes
- FortiAnalyzer integration options
- Initial configuration, default ports, and security baseline
- First-login setup wizard
- Administrator accounts and profiles
- Concurrent administrators — workspace and workflow modes
- ADOM best practices and upgrade order
- Backup, restore, migration, and offline mode
- Provisioning templates and normalized interfaces
- Device revision history vs ADOM revisions
- ADOM-level vs device-level database split
- Scripting targets
- FortiManager behind NAT (`mgmt-addr`)

## Build / run

It is a single static HTML file with no dependencies. Open `index.html`
locally, or browse the live site link above.
